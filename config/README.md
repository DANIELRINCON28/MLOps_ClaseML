# Carpeta config

Archivos de configuración del proyecto.

## Contenido:

- **config.json**: Configuración general del proyecto (rutas, parámetros, etc.)
- **pytest.ini**: Configuración de pytest para tests unitarios
- **sonar-project.properties**: Configuración de SonarCloud para análisis de calidad de código
- **Dockerfile**: Configuración para contenedorización con Docker
- **docker-compose.yml**: Orquestación de servicios Docker

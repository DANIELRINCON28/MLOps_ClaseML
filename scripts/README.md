# Carpeta scripts

Scripts de ejecución y utilidades del proyecto.

## Contenido:

- **run_all.ps1**: Script PowerShell para ejecutar el pipeline completo (Windows)
- **run_all.sh**: Script Bash para ejecutar el pipeline completo (Linux/Mac)
- **ejecutar_mlops.bat**: Batch script para Windows
- **set_up.bat**: Script de configuración inicial
- **check_environment.py**: Verifica el entorno de Python
- **check_data.py**: Valida los datos de entrada
